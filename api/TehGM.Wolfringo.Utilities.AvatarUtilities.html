    Parameters
    Type Name Description
    WolfGroup group

    Group to get avatar URL for.
    UInt32 size

    Size of the avatar to retrieve.
    Returns
    Type Description
    String

    URL to group's avatar.
    Remarks

    This method will return URL to group avatar. You can use this URL with HttpClient to download the avatar, or use for any other purpose.

    size is only a requested size of the avatar. It is not guaranteed that URL will contain avatar with this size.

    Nullable<Int32> iconID

    Icon ID
    UInt32 size

    Size of the avatar to retrieve.
    Returns
    Type Description
    String

    URL to group's avatar.
    Remarks

    This method will return URL to group avatar. You can use this URL with HttpClient to download the avatar, or use for any other purpose.

    size is only a requested size of the avatar. It is not guaranteed that URL will contain avatar with this size.
